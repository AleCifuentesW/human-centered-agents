# Human-Centered Agents

AI agents should not only be powerful. They should be clear, useful, calm, and kind to the human mind.

Los agentes de IA no solo deberían ser poderosos. También deberían ser claros, útiles, tranquilos y cuidadosos con la mente humana.

This open-source project helps people learn prompts and design human-centered agents step by step, with a focus on clarity, cognitive load, and anti-overwhelm design.

Este proyecto open-source ayuda a aprender prompts y diseñar agentes centrados en humanos paso a paso, con foco en claridad, carga cognitiva y diseño anti-saturación.
