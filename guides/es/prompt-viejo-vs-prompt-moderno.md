# Prompt viejo vs prompt moderno

## Antes (prompt viejo)
Era común escribir cosas como:
- "Actúa como experto en X".

Eso a veces funciona, pero suele quedar genérico.

## Ahora (prompt moderno)
Suele funcionar mejor definir:
- contexto
- objetivo
- flujo de trabajo (workflow)
- límites
- formato de respuesta

## Ejemplo
- Viejo: "Actúa como experto y ayúdame con mi negocio".
- Moderno: "Tengo una tienda pequeña de ropa en Valparaíso. Dame 3 acciones para aumentar ventas este mes, con presupuesto bajo, y entrégalo en tabla con prioridad alta/media/baja".

## Idea clave
Hoy no gana el prompt más "elegante". Gana el más claro y accionable.
