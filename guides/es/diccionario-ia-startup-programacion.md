# Diccionario IA + Startup + Programación

Guía base de términos que vamos a usar en este repo.
Formato por entrada:
- sigla o término
- significado original (inglés)
- traducción o equivalencia en español
- explicación simple
- ejemplo corto

## IA y prompts

### IA
- Inglés: Artificial Intelligence
- Equivalencia en español: inteligencia artificial
- En simple: tecnología que puede analizar información y generar respuestas o acciones útiles.
- Ejemplo: "Uso IA para ordenar ideas y definir mi siguiente paso."

### Prompt
- Inglés: Prompt
- Equivalencia en español: instrucción o pedido
- En simple: instrucción o pedido que le haces a la IA.
- Ejemplo: "Resume este texto en 5 líneas".

### LLM
- Inglés: Large Language Model
- Equivalencia en español: modelo de lenguaje grande
- En simple: modelo entrenado con mucho texto para entender y generar lenguaje.
- Ejemplo: "Le pedí al LLM que me explicara GitHub en palabras simples".

### Contexto
- Inglés: Context
- Equivalencia en español: contexto
- En simple: información previa que ayuda a la IA a responder mejor.
- Ejemplo: "Contexto: tengo una tienda de barrio y presupuesto bajo".

### Workflow
- Inglés: Workflow
- Equivalencia en español: flujo de trabajo
- En simple: secuencia de pasos para resolver una tarea.
- Ejemplo: "Primero resume, luego prioriza, después propone un siguiente paso".

### Alucinación
- Inglés: Hallucination
- Equivalencia en español: alucinación
- En simple: cuando la IA inventa datos como si fueran reales.
- Ejemplo: "La IA citó una fuente que no existía: eso es alucinación".

### Tokens
- Inglés: Tokens
- Equivalencia en español: tokens (unidades de texto)
- En simple: piezas de texto que usa el modelo para procesar entradas y salidas.
- Ejemplo: "Si el prompt es muy largo, consume más tokens".

## Startup

### MVP
- Inglés: Minimum Viable Product
- Equivalencia en español: producto mínimo viable
- En simple: versión mínima de un producto para probar si tiene valor real.
- Ejemplo: "Antes de crear la app completa, hicimos un MVP con una sola función".

### PMF
- Inglés: Product-Market Fit
- Equivalencia en español: encaje producto-mercado
- En simple: cuando el producto calza bien con una necesidad real del mercado.
- Ejemplo: "La gente volvió a usarlo y recomendó el producto: señal de PMF".

### ICP
- Inglés: Ideal Customer Profile
- Equivalencia en español: perfil de cliente ideal
- En simple: tipo de cliente ideal al que más le sirve tu producto.
- Ejemplo: "Nuestro ICP son pymes de 5 a 20 personas".

### CAC
- Inglés: Customer Acquisition Cost
- Equivalencia en español: costo de adquisición de cliente
- En simple: costo promedio para conseguir un cliente nuevo.
- Ejemplo: "Si gastas 100 y llegan 10 clientes, el CAC es 10".

### LTV
- Inglés: Lifetime Value
- Equivalencia en español: valor de vida del cliente
- En simple: valor total que deja un cliente durante su relación con tu negocio.
- Ejemplo: "Un cliente que paga mensual por 2 años tiene mayor LTV".

### Runway
- Inglés: Runway
- Equivalencia en español: pista financiera
- En simple: meses que la empresa puede seguir operando con el dinero disponible.
- Ejemplo: "Con caja actual, tenemos 8 meses de runway".

## Programación

### API
- Inglés: Application Programming Interface
- Equivalencia en español: interfaz de programación de aplicaciones
- En simple: puente para que dos sistemas se comuniquen.
- Ejemplo: "La app usa una API para traer datos del clima".

### Backend
- Inglés: Backend
- Equivalencia en español: capa de servidor / parte interna
- En simple: parte interna del sistema (lógica, base de datos, servidores).
- Ejemplo: "El backend valida usuarios y guarda pedidos".

### Frontend
- Inglés: Frontend
- Equivalencia en español: interfaz / capa visual
- En simple: parte visual con la que interactúa la persona usuaria.
- Ejemplo: "El frontend muestra el carrito y el botón de pago".

### Deploy
- Inglés: Deployment
- Equivalencia en español: despliegue
- En simple: publicar una aplicación para que otras personas la usen.
- Ejemplo: "Después de probar localmente, hicimos deploy a producción".

### Repo
- Inglés: Repository
- Equivalencia en español: repositorio
- En simple: carpeta/proyecto versionado con Git.
- Ejemplo: "Subimos los cambios al repo para revisión".

### Pull Request (PR)
- Inglés: Pull Request
- Equivalencia en español: solicitud de cambios
- En simple: propuesta de cambio en código para que el equipo la revise.
- Ejemplo: "Abrí un PR con el fix y pedí feedback".

### Overselling
- Inglés: Overselling
- Equivalencia en español: sobreventa
- En simple: prometer más de lo que el producto o la IA realmente puede cumplir.
- Ejemplo: "Decir que tu MVP resolverá todo desde el día uno es overselling".

### Oops Loop
- Inglés: Oops Loop
- Equivalencia en español: ciclo de errores repetidos
- En simple: patrón donde se repite el mismo error porque no se detiene para revisar y ajustar.
- Ejemplo: "Lanzar, fallar y relanzar sin aprendizaje claro puede generar un oops loop".

### Feedback
- Inglés: Feedback
- Equivalencia en español: retroalimentación
- En simple: comentarios o señales que te ayudan a mejorar una idea, producto o respuesta.
- Ejemplo: "El feedback de 5 usuarios cambió nuestra propuesta de valor".

### Vibe Check
- Inglés: Vibe Check
- Equivalencia en español: chequeo rápido de sensación
- En simple: revisión breve para validar si algo se siente claro, coherente y útil antes de seguir.
- Ejemplo: "Hagamos un vibe check del pitch antes de enviarlo".

### Markdown
- Inglés: Markdown
- Equivalencia en español: formato de marcado ligero
- En simple: forma simple de dar estructura a texto con títulos, listas, enlaces y código.
- Ejemplo: "Este README está escrito en Markdown".

### GitHub
- Inglés: GitHub
- Equivalencia en español: plataforma de colaboración con Git
- En simple: plataforma para alojar repositorios, colaborar en código y revisar cambios.
- Ejemplo: "Subimos el proyecto a GitHub para trabajar en equipo".

### Resources
- Inglés: Resources
- Equivalencia en español: recursos
- En simple: materiales de apoyo como guías, plantillas, enlaces, datasets o herramientas.
- Ejemplo: "Partamos con tres resources clave para aprender sin saturarte".

## Nota de uso
Este diccionario está en versión inicial y se irá ampliando con nuevos términos a medida que avance el proyecto.
