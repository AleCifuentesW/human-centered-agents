# Cómo usar un agente

Un agente funciona mejor cuando le das dirección clara.

## Plantilla rápida
1. Contexto: qué está pasando.
2. Objetivo: qué resultado necesitas.
3. Límites: tiempo, presupuesto, nivel técnico.
4. Formato: cómo quieres la respuesta.

## Ejemplo
"Quiero ordenar una idea de emprendimiento. Tengo 2 horas semanales y cero equipo. Dame solo el siguiente paso recomendado y por qué".

## Consejo práctico
Si te abruma la respuesta, pide:
"Resúmelo en una sola prioridad para hoy".
