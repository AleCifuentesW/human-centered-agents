# Qué es un prompt

Un prompt no es magia.
Es una forma ordenada de pedir ayuda a una IA.

Si quieres una frase fácil de recordar:
**prompt = chamullo ordenado que le dices a la IA**.

No se trata de sonar "pro". Se trata de ser claro.

## Ejemplo simple
- Prompt débil: "Háblame de marketing".
- Prompt útil: "Explícame en 5 puntos cómo mejorar el marketing de una cafetería de barrio, con ideas baratas y ejemplos concretos".

## Idea clave
Mientras más claro sea el pedido, más útil suele ser la respuesta.
