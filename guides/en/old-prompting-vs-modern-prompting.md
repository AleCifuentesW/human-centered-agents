# Old prompting vs modern prompting

## Before (old prompting)
A common style was:
- "Act as an expert in X".

It can work, but outputs are often generic.

## Now (modern prompting)
It is usually better to define:
- context
- goal
- workflow
- boundaries
- response format

## Example
- Old: "Act as an expert and help my business".
- Modern: "I run a small clothing shop. Give me 3 low-budget actions to increase sales this month, and present them in a table with high/medium/low priority".

## Key idea
Today, the best prompt is not the fanciest one. It is the clearest actionable one.
