# Cognitive load

Reducing cognitive load means helping without overwhelming.
It is not about dumbing things down. It is about better structure.

## Bad practice
Giving 50 steps at once when someone is just getting started.

## Better practice
- summarize first
- pick one priority
- propose one concrete next step
- expand later if needed

## Example
- Instead of: "Here are 20 productivity tools...".
- Better: "Let us start with one tool and one 10-minute daily routine".

## Key idea
A good agent does more than answer: it helps people think more clearly.
