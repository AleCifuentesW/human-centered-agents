# AI + Startup + Programming Glossary

Base glossary of terms used in this repository.
Entry format:
- term or acronym
- original meaning (English)
- Spanish translation or equivalent
- plain explanation
- short example

## AI and prompting

### Prompt
- English meaning: Prompt
- Spanish equivalent: instruccion o pedido
- Plain explanation: the instruction you give the AI.
- Example: "Summarize this text in 5 lines."

### LLM
- English meaning: Large Language Model
- Spanish equivalent: modelo de lenguaje grande
- Plain explanation: a model trained on large text datasets to understand and generate language.
- Example: "I asked the LLM to explain GitHub in simple words."

### Context
- English meaning: Context
- Spanish equivalent: contexto
- Plain explanation: background information that helps the AI respond better.
- Example: "Context: I run a small shop with a low budget."

### Workflow
- English meaning: Workflow
- Spanish equivalent: flujo de trabajo
- Plain explanation: a sequence of steps to complete a task.
- Example: "First summarize, then prioritize, then propose one next step."

### Hallucination
- English meaning: Hallucination
- Spanish equivalent: alucinacion
- Plain explanation: when the AI invents information and presents it as factual.
- Example: "The AI cited a source that does not exist: that is a hallucination."

### Tokens
- English meaning: Tokens
- Spanish equivalent: tokens (unidades de texto)
- Plain explanation: text units the model uses to process input and output.
- Example: "A longer prompt usually uses more tokens."

## Startup

### MVP
- English meaning: Minimum Viable Product
- Spanish equivalent: producto minimo viable
- Plain explanation: the smallest product version used to test real value.
- Example: "Before building the full app, we launched an MVP with one core feature."

### PMF
- English meaning: Product-Market Fit
- Spanish equivalent: encaje producto-mercado
- Plain explanation: when a product strongly matches a real market need.
- Example: "Users kept returning and recommending it: a PMF signal."

### ICP
- English meaning: Ideal Customer Profile
- Spanish equivalent: perfil de cliente ideal
- Plain explanation: the customer type that benefits most from your product.
- Example: "Our ICP is small teams of 5 to 20 people."

### CAC
- English meaning: Customer Acquisition Cost
- Spanish equivalent: costo de adquisicion de cliente
- Plain explanation: average cost to acquire one new customer.
- Example: "If you spend 100 and get 10 customers, CAC is 10."

### LTV
- English meaning: Lifetime Value
- Spanish equivalent: valor de vida del cliente
- Plain explanation: total value a customer generates over time.
- Example: "A customer paying monthly for two years has higher LTV."

### Runway
- English meaning: Runway
- Spanish equivalent: pista financiera
- Plain explanation: how many months the company can operate with current cash.
- Example: "With our current cash, we have 8 months of runway."

## Programming

### API
- English meaning: Application Programming Interface
- Spanish equivalent: interfaz de programacion de aplicaciones
- Plain explanation: a bridge that lets systems communicate.
- Example: "The app uses an API to fetch weather data."

### Backend
- English meaning: Backend
- Spanish equivalent: capa de servidor / parte interna
- Plain explanation: internal system layer (logic, database, servers).
- Example: "The backend validates users and stores orders."

### Frontend
- English meaning: Frontend
- Spanish equivalent: interfaz / capa visual
- Plain explanation: the visual layer users interact with.
- Example: "The frontend shows the cart and checkout button."

### Deploy
- English meaning: Deployment
- Spanish equivalent: despliegue
- Plain explanation: publishing an app so people can use it.
- Example: "After local testing, we deployed to production."

### Repo
- English meaning: Repository
- Spanish equivalent: repositorio
- Plain explanation: a Git-versioned project folder.
- Example: "We pushed changes to the repo for review."

### Pull Request (PR)
- English meaning: Pull Request
- Spanish equivalent: solicitud de cambios
- Plain explanation: a proposed code change for team review.
- Example: "I opened a PR with the fix and requested feedback."

## Usage note
