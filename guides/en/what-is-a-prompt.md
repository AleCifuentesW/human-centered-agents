# What is a prompt

A prompt is not magic.
It is a structured way to ask an AI for help.

You do not need fancy wording. You need clarity.

## Simple example
- Weak prompt: "Tell me about marketing".
- Useful prompt: "Explain in 5 points how to improve marketing for a small neighborhood coffee shop, using low-cost ideas and concrete examples".

## Key idea
The clearer your request is, the more useful the answer tends to be.
