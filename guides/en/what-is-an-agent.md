# What is an agent?

An agent is an AI with stable instructions so it can help you more consistently.

If a prompt is a one-time request,
an agent is a repeatable way of working.

## In simple terms
- Prompt: "do this now".
- Agent: "when I ask for help, respond this way".

## What a good agent includes
- clear purpose
- when to use it and when not to
- workflow
- boundaries
- response format
- quality criteria

## Concrete example
- Repeated ask: "Organize my ideas, do not overwhelm me, and give me one concrete next step".
- Solution: use an agent already configured for that behavior.

## Key idea
A good agent does more than answer questions.
It helps you think with more clarity and less mental load.
