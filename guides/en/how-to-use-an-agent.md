# How to use an agent

An agent is an AI system with clearer instructions than a regular chat.

It is not magic, and it is not a real person.
It is a way to tell AI:

> "Help me with this type of problem, using this way of working, with these limits, and this response style."

The difference between using a general AI chat and using an agent is that the agent already has a defined way to think and respond.

For example:

- an idea-organizing agent should not give you 50 options at once
- a startup validation agent should challenge your assumptions
- a learning agent should explain things step by step
- a writing agent should protect tone, clarity, and structure

## What an agent needs to help you well

An agent usually works better when you provide 4 things:

### 1. Context

Explain what is happening.

Example:

> I have a startup idea, but I do not know where to start.

### 2. Goal

Say what outcome you need.

Example:

> I want to know what my first step should be.

### 3. Constraints

Share your limitations.

Example:

> I have little time, no team, and I do not want to spend money yet.

### 4. Format

Tell the agent how you want the response.

Example:

> Give me only one priority and one concrete next step.

## Simple template

You can copy and fill this:

```txt
Context:
[What is happening]

Goal:
[What you need to achieve]

Constraints:
[Time, budget, technical level, energy, or other limits]

Format:
[How you want the response]
```

## Full example

```txt
Context:
I want to organize a startup idea, but I have too many things in my head.

Goal:
I need to know where to begin.

Constraints:
I only have 2 hours per week, no team, and I do not want to spend money yet.

Format:
Give me only one main priority, one concrete next step, and a short explanation of why this step comes first.
```

## What to do if the response feels overwhelming

If the agent answers with too much detail, you can ask:

> Summarize this into one priority for today.

Or:

> Do not give me more options yet. Tell me only the next step.

## What to do if the response is too general

If the response feels too broad, you can say:

> Make it more concrete. Give me one action I can do in less than 30 minutes.

## What to do if you do not know what to ask

You can start like this:

> I am not sure how to explain this well. Help me organize the problem first before giving solutions.

## Important rule

A good agent should not make you feel more lost.

If after using it you feel more confusion, more tasks, and more anxiety, the agent is not doing its job well.

In this project, agents are designed to help you move step by step, with less noise and more clarity.
