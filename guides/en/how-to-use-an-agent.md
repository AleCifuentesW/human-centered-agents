# How to use an agent

An agent works better when you provide clear direction.

## Quick template
1. Context: what is happening.
2. Goal: what outcome you need.
3. Constraints: time, budget, technical level.
4. Format: how you want the response.

## Example
"I want to organize a startup idea. I only have 2 hours per week and no team. Give me only the recommended next step and why".

## Practical tip
If the response feels heavy, ask:
"Summarize this into one priority for today".
